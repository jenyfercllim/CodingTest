User Guideline
==============
1. Click on RunDemo.bat to compile and launch application.
2. CSV file is under CodingTest\src\main\resources\game_sales.csv
3. Visit http://localhost:8080/import to upload CSV file. There is "Next" button will redirect to /getGameSales page.
4. Visit http://localhost:8080/getGameSales to retrieve list of game sales.
